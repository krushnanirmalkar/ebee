How It Works

Smart DB is the intelligent heart of your EV charging network.
It connects, controls and optimizes every charging point seamlessly.

User Arrives

User parks the vehicle and plugs in the charger to the 16A socket.

Initiate Charging

User scans the QR code / opens the app and selects the bay.

Make Payment

User makes the payment through preferred payment mode.

Payment Verification

Payment is verified securely on the backend server.

Command to Smart DB

Backend sends command to Smart DB to activate the selected 16A socket.

Current Allowed

Smart DB allows electricity to the socket and charging starts.

Load Distribution

If multiple cars are charging, Smart DB intelligently distributes available current across all bays.
